# Student Portal Routing React

This is a React project demonstrating **routing with React Router**.  
The project includes pages like **Home, Profile, Courses, Contact**, and a **Navbar** for navigation.

## Features
- Single Page Application (SPA)
- React Router v6 for navigation
- Functional Components
